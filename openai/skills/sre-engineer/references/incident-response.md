# Incident Response

Use this guide to structure incident command and escalation.

## Severity Model

Define severity levels tied to impact and urgency.

## Command Roles

For high-severity incidents define:
- Incident commander
- Operations/mitigation lead
- Communications lead
- Subject matter responders

## Escalation and Communication

Define:
- Escalation triggers
- Stakeholder communication cadence
- Handoff protocol across shifts/time zones

## Incident Lifecycle

A complete lifecycle includes:
- Detection
- Triage
- Stabilization
- Recovery
- Closure with postmortem follow-up
