# IaC Governance

Use this guide to govern infrastructure change safely.

## Governance Baseline

Define:
- IaC source-of-truth repository boundaries
- Mandatory review and approval policy
- Policy-as-code checks and enforcement points

## Drift Management

Define:
- Drift detection cadence/signals
- Ownership for remediation
- Emergency change reconciliation workflow

## Auditability

Ensure:
- Traceability from change request to deployed state
- Immutable change logs for regulated systems
- Exception records with expiry and owner
