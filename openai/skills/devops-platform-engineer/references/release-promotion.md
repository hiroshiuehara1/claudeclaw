# Release Promotion

Use this guide for environment promotion and deployment safety.

## Promotion Model

Define:
- Promotion stages and entry criteria
- Required checks for each stage
- Approval responsibilities

## Deployment Strategies

Choose based on risk:
- Canary
- Blue/green
- Rolling
- Shadow

## Rollback Governance

For each strategy, define:
- Objective rollback triggers
- Automation path
- Post-rollback verification checks
